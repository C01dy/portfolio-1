Немного о себе 🙂

Увлекаюсь программированием, веб-разработкой заинтересовался примерно в августе 2019 года. Сперва изучал простую верстку на HTML и CSS, в сентябре начал изучать JavaScript. На данный момент изучил React в связке Redux, основы платформы nodeJS, аспекты работы сборки проектов на React с использованием webpack и babel. На профиле в github имеются небольшие пет проекты, написанные с использованием выше озвученных технологий. Имею желание изучать другие технологии и научиться работать в команде. Сейчас моя цель - начать карьеру веб-разработчика, получать непрерывный опыт, развивать свои навыки, чтобы в дальнейшем повышать свою квалификацию. Сейчас обучаюсь в ЮРГПУ (НПИ) по направлению "Прикладная математика" на втором курсе. Всё что касается уже именно веб-разработки - изучаю самостоятельно.
